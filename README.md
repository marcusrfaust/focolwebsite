# Projeto de Redesign do Site Focol

**Data:** 7 de maio de 2025  
**Versão:** 1.1  
**Status:** Em desenvolvimento

## Visão Geral

Este documento contém o plano estratégico completo para o redesign do site da Focol (www.focol.com.br), uma plataforma omnichannel de atendimento e vendas que unifica canais como WhatsApp, Instagram, Messenger, SMS, Telegram e outros em um único sistema.

O objetivo é transformar o site atual em uma experiência premium que transmita autoridade, inovação e escalabilidade, competindo diretamente com soluções como Chatwoot e Blip, mas destacando os diferenciais de liberdade, controle e personalização da Focol.

## Análise do Site Atual

O site atual da Focol apresenta:
- Informações básicas sobre recursos (chatbots, atendimento ao cliente, relatórios)
- Depoimentos de clientes (diretores e gerentes)
- Seção de preços com alternância mensal/anual
- Seção de perguntas frequentes
- Múltiplos CTAs ("Comece agora!" e "Já quero comprar!")

Oportunidades de melhoria:
- Elevação do posicionamento visual para transmitir valor premium
- Narrativa mais clara sobre a proposta de valor omnichannel
- Estrutura de navegação mais intuitiva e fluida
- Design que demonstre melhor as capacidades tecnológicas da plataforma

## Estratégia de Redesign

### 1. Identidade Visual e Branding

#### Paleta de Cores:
- **Primárias:** Preto profundo (#0A0A0A) e branco nítido (#FFFFFF) como cores base
- **Secundárias:** Toques estratégicos de roxo profundo (#5E17EB) e azul elétrico (#0066FF)
- **Terciárias:** Vários tons de cinza (#F8F8F8, #E0E0E0, #B0B0B0, #606060)

#### Tipografia:
- **Títulos:** Montserrat (Bold) para declarações fortes e autoritárias
- **Corpo:** Inter (Regular, Medium) para legibilidade limpa
- **Micro-textos:** Inter (Light) para elementos sutis da interface

#### Elementos Visuais:
- Sobreposições de gradiente sutis para profundidade (preto para roxo/azul profundo)
- Iconografia minimalista com linhas finas e precisas
- Micro-animações para elementos interativos (botões, toggles, menus dropdown)
- Capturas de tela de alta qualidade com mockups de dispositivos mostrando a plataforma em ação

### 2. Arquitetura do Site

#### Página Inicial:
- **Seção Hero:**
  - Título impactante: "Unifique. Automatize. Escale."
  - Subtítulo: "Centralize todos os canais em uma única plataforma e transforme atendimentos em vendas"
  - Ilustração animada mostrando mensagens de diferentes canais fluindo para uma interface unificada
  - CTA primário: "Agende uma demonstração"
  - CTA secundário: "Teste grátis por 14 dias"
  - Seletor de idioma flutuante (PT/ES/EN) no canto superior direito

#### Seções de Funcionalidades:

1. **Integração Omnichannel**
   - Visual mostrando múltiplos canais convergindo
   - Animação de transferência perfeita de mensagens entre plataformas
   - Descrição: "Todos os seus canais em um só lugar, sem complicações"

2. **Automação com IA**
   - Demo interativo de criação de chatbot
   - Representação visual de automação de fluxo de trabalho
   - Descrição: "Automatize sem perder a personalização"

3. **CRM e Analytics**
   - Prévias de dashboard limpas e minimalistas
   - Exemplos animados de visualização de dados
   - Descrição: "Dados que transformam conversas em conversões"

4. **Gestão de Equipes**
   - Visual de múltiplos agentes gerenciando conversas
   - Ilustração de monitoramento em tempo real
   - Descrição: "Controle total sobre sua operação de atendimento"

#### Seção "Para Quem":
- Três seções distintas para:
  - **Empresas:** Foco em escalabilidade e integração com sistemas existentes
  - **Agências:** Ênfase em gerenciamento de múltiplos clientes e white-label
  - **Prestadores de Serviço:** Destaque para automação e eficiência de atendimento
- Cada um com casos de uso específicos e benefícios personalizados

#### Preços:
- Tabela de preços premium e transparente
- Comparação de recursos com indicadores visuais
- Ênfase nas opções de personalização
- Plano Enterprise com CTA "Contacte-nos"

#### Prova Social:
- Showcase minimalista de logos de clientes
- Depoimentos com fotos profissionais
- Prévias de estudos de caso com resultados mensuráveis

#### Blog/Recursos:
- Centro de conhecimento com conteúdo categorizado
- Materiais premium para download (whitepapers, guias)
- Inscrição na newsletter com campos mínimos

#### Contato/Demonstração:
- Formulário simplificado (nome, e-mail, empresa, telefone)
- Integração de calendário para agendamento direto
- Botão WhatsApp para contato imediato

### 3. Melhorias de UX

#### Navegação:
- Navbar minimalista fixo que se transforma ao rolar
- Mega menu para navegação profunda com indicadores visuais
- Menu hamburger otimizado para mobile com transições suaves

#### Velocidade e Performance:
- Lazy loading para imagens e animações
- Carregamento progressivo para seções de conteúdo
- Base de código otimizada para carregamento em menos de 2 segundos

#### Otimização de Conversão:
- Posicionamento estratégico de CTAs ao longo da jornada do usuário
- Popup de intenção de saída oferecendo conteúdo valioso
- Campos de formulário inteligentes com preenchimento automático
- Botão flutuante do WhatsApp para contato direto

#### Acessibilidade:
- Conformidade com WCAG 2.1 AA
- Toggle para modo de alto contraste
- Suporte para navegação por teclado
- Otimização para leitores de tela

### 4. Estratégia de Texto

#### Tom e Voz:
- Autoritário, mas acessível
- Focado em soluções, não apenas em recursos
- Ênfase em ROI e transformação de negócios

#### Mensagens-Chave:
- "Controle total sobre seus canais de comunicação"
- "Automatize sem perder a personalização"
- "Escale seu atendimento com inteligência"
- "Dados que transformam conversas em conversões"

#### Suporte Multilíngue:
- Sistema completo de tradução para português, espanhol e inglês
- Adaptações de conteúdo específicas para cada região
- Respeito às nuances culturais em cada idioma

### 5. Recomendações Técnicas

#### Framework de Desenvolvimento:
- Next.js para benefícios de performance e SEO (configurar para exportação estática para GitHub Pages, se necessário)
- React para arquitetura baseada em componentes
- Framer Motion para animações premium

#### Integração de CMS:
- CMS Headless (Contentful ou Sanity) para fácil gerenciamento de conteúdo
- Sistema de localização para suporte multilíngue

#### Analytics e Rastreamento:
- Google Tag Manager para rastreamento flexível
- Integração de mapas de calor (Hotjar)
- Capacidade de teste A/B (Google Optimize)

## Critérios de Sucesso

- MVP: Presença online funcional em até 2 semanas, com landing page informativa e responsiva
- MLP: Site com experiência visual atraente e navegação fluida em até 4 semanas
- Versão Completa: Experiência premium completa em até 8 semanas
- O site deve transmitir uma experiência premium, mas acessível, com foco em conversão de leads qualificados
- O design deve refletir a identidade da Focol como uma solução confiável, inovadora e escalável
- A navegação deve ser intuitiva, com tempo de carregamento abaixo de 2 segundos e compatibilidade total com dispositivos móveis
- As copys devem gerar identificação com o público-alvo (empresas e agências) e incentivar a ação (demonstração ou teste gratuito)
