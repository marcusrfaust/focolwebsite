# Guia de Estilo - Focol

## Paleta de Cores

### Cores Primárias
- **Preto Profundo:** #0A0A0A
- **Branco Nítido:** #FFFFFF

### Cores Secundárias
- **Roxo Profundo:** #5E17EB
- **Azul Elétrico:** #0066FF

### Cores Terciárias
- **Cinza Claro:** #F8F8F8
- **Cinza Médio-Claro:** #E0E0E0
- **Cinza Médio:** #B0B0B0
- **Cinza Escuro:** #606060

## Tipografia

### Títulos
- **Fonte:** Montserrat Bold
- **Uso:** Headings H1, H2, H3
- **Pesos:** 700 (Bold)

### Corpo
- **Fonte:** Inter
- **Uso:** Parágrafos, listas, menus de navegação
- **Pesos:** 400 (Regular), 500 (Medium)

### Micro-textos
- **Fonte:** Inter Light
- **Uso:** Legendas, rodapés, microcópia da interface
- **Pesos:** 300 (Light)

## Botões

### Botão Primário
- **Background:** Gradiente de #5E17EB para #4A00E0
- **Texto:** #FFFFFF
- **Border-radius:** 8px
- **Padding:** 12px 24px
- **Hover:** Leve efeito de zoom (1.05) com transição suave

### Botão Secundário
- **Background:** Transparente
- **Borda:** 2px solid #5E17EB
- **Texto:** #5E17EB
- **Border-radius:** 8px
- **Padding:** 12px 24px
- **Hover:** Background #5E17EB15 com transição suave

### Botão Terciário
- **Background:** Transparente
- **Texto:** #5E17EB
- **Border-radius:** 8px
- **Padding:** 12px 24px
- **Hover:** Underline com transição suave

## Iconografia

- Estilo minimalista com linhas finas (1.5px)
- Cantos levemente arredondados
- Tamanho padrão: 24x24px para navegação, 20x20px para ações em linha
- Cores: Adaptadas à paleta (principalmente preto, cinza escuro ou roxo profundo)

## Elementos de Interface

### Cards
- **Background:** #FFFFFF
- **Sombra:** 0px 8px 24px rgba(0, 0, 0, 0.06)
- **Border-radius:** 12px
- **Padding:** 24px
- **Hover:** Elevação sutil (elevação da sombra)

### Input Fields
- **Background:** #F8F8F8
- **Borda:** 1px solid #E0E0E0
- **Border-radius:** 8px
- **Padding:** 12px 16px
- **Focus:** Borda #5E17EB com transição suave

### Dropdowns
- **Background:** #FFFFFF
- **Borda:** 1px solid #E0E0E0
- **Border-radius:** 8px
- **Sombra:** 0px 4px 12px rgba(0, 0, 0, 0.08)

## Animações e Microinterações

### Transições
- **Duração:** 0.2s para microinterações, 0.3-0.4s para animações maiores
- **Timing-function:** ease-in-out
- **Propriedades animadas:** opacity, transform, background-color

### Hover States
- Transições suaves para todos os elementos interativos
- Indicações visuais sutis (elevação, mudança de tom, zoom leve)

### Animações de Página
- Entrada de elementos com fade-in suave (staggered)
- Parallax sutil em elementos de destaque
- Transições de página com dissolve

## Espaçamento e Grid

### Sistema de Grid
- Base de 12 colunas para desktop
- Margens responsivas (64px desktop, 32px tablet, 16px mobile)
- Gutters: 24px (desktop), 16px (tablet), 8px (mobile)

### Sistema de Espaçamento
- Base em múltiplos de 8px
- 8px (XS), 16px (S), 24px (M), 32px (L), 48px (XL), 64px (XXL)

## Tratamento de Imagens

### Estilo Fotográfico
- Alto contraste
- Tons modernos e tecnológicos
- Foco em ambientes profissionais modernos
- Pessoas diversas usando a plataforma em contextos reais

### Ilustrações
- Estilo clean e minimalista
- Paleta coordenada com as cores da marca
- Linhas finas e precisas
- Representações abstratas de conceitos tecnológicos
