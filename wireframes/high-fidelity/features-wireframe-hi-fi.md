## Layout Mobile (375px)

### Adaptações para Responsividade
- **Header:** Menu hamburger para navegação em tela pequena
- **Seções de Funcionalidades:** 
  - Layout em uma coluna para todas as seções
  - Visual sempre acima do texto
  - Redução do tamanho dos títulos (36px → 28px)
  - Ajuste de padding (96px 64px → 64px 24px)
- **CTA Final:** Botões empilhados
- **Footer:** Seções empilhadas com acordeão para as categorias

## Interações Específicas

1. **Navegação por tab:**
   - Tabs na parte superior para navegar rapidamente entre as funcionalidades
   - Indicador de progresso (1/4, 2/4, etc.)
   - Scroll automático para a seção correspondente

2. **Demonstrações interativas:**
   - Vídeos curtos (15s) em loop mostrando cada funcionalidade
   - Interruptor para alternar entre vídeo e imagem estática
   - Opção de expandir para visualização em tela cheia

3. **Comparativo de recursos:**
   - Toggle para ver comparação detalhada com concorrentes
   - Matriz de recursos com indicação visual de diferenciais

4. **Transições entre seções:**
   - Fade in/out com parallax sutil ao rolar
   - Numeração das seções com efeito de contagem ao entrar no viewport

## Especificações Técnicas

- **Framework:** Next.js
- **Animações:** Framer Motion para transições e interações
- **Responsividade:** Breakpoints em 1440px, 1024px, 768px, 375px
- **Performance:** Carregamento progressivo das seções (prioridade para conteúdo visível)
- **Acessibilidade:** Estrutura semântica, contraste apropriado, navegação por teclado
